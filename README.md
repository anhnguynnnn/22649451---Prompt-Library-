# 📚 Prompt Library

This repository contains a structured prompt library developed for workflow automation in a wholesale fashion business context.

---

## 🏢 Business Context

This prompt library is based on **Dorsia**, a wholesale fashion company that works with:

- fashion boutiques across Melbourne
- fashion brands that want to partner with Dorsia
- internal teams managing inquiries, communication, and follow-up workflows

The purpose of this library is to show how AI prompts can support operational efficiency, consistency, and faster response handling in real business workflows.
